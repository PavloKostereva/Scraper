What is Canvas Fingerprinting?

Canvas fingerprinting is a technique that uses the HTML5 Canvas API to identify browsers. Different browsers (and especially headless browsers) render text and graphics slightly differently due to:

  • Different rendering engines (Blink, Gecko, WebKit)
  • Different font rendering algorithms
  • Different graphics drivers
  • Different operating systems

Headless browsers often have distinct, consistent rendering patterns that make them easy to identify.
